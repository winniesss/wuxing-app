# 快速添加课程指南

## 📝 课程数据结构

每个课程包含以下字段：
- `id`: 课程编号（数字，从1开始递增）
- `title`: 课程标题
- `description`: 课程描述
- `questions`: 题目数组

## 🎯 题目类型

### 1. 单选题 (single_choice)

```javascript
{
  id: 1,  // 题目编号（从1开始）
  type: "single_choice",
  question: "问题内容？",
  options: ["选项1", "选项2", "选项3", "选项4"],
  correctAnswer: "选项1",  // 正确答案
  explanation: "解释说明"
}
```

### 2. 填空题 (fill_in_blank)

```javascript
{
  id: 2,
  type: "fill_in_blank",
  question: "填空题内容：__、__、__。",
  correctAnswer: "答案1,答案2,答案3",  // 多个答案用逗号分隔
  explanation: "解释说明"
}
```

## 🚀 快速添加步骤

### 方法一：直接在 data.js 中添加

1. **打开文件**：`src/data.js`

2. **找到课程数组末尾**，在最后一个课程后面添加新课程：

```javascript
export const lessons = [
  // ... 现有课程 ...
  {
    id: 8,  // 下一个课程编号
    title: "新课程标题",
    description: "课程描述",
    questions: [
      {
        id: 1,
        type: "single_choice",
        question: "问题1？",
        options: ["选项A", "选项B", "选项C", "选项D"],
        correctAnswer: "选项A",
        explanation: "解释说明"
      },
      {
        id: 2,
        type: "fill_in_blank",
        question: "填空题：__、__。",
        correctAnswer: "答案1,答案2",
        explanation: "解释说明"
      }
      // 可以添加更多题目...
    ]
  }
];
```

3. **保存文件**，刷新页面即可看到新课程

### 方法二：使用模板快速复制

复制以下模板，修改内容即可：

```javascript
{
  id: 8,  // 修改为下一个编号
  title: "你的课程标题",
  description: "课程简短描述",
  questions: [
    {
      id: 1,
      type: "single_choice",
      question: "问题内容？",
      options: ["选项1", "选项2", "选项3", "选项4"],
      correctAnswer: "选项1",
      explanation: "正确答案的解释"
    },
    {
      id: 2,
      type: "fill_in_blank",
      question: "填空题：__、__、__。",
      correctAnswer: "答案1,答案2,答案3",
      explanation: "答案的解释说明"
    }
  ]
}
```

## 📋 添加示例

### 示例：添加"十二地支配对"课程

```javascript
{
  id: 8,
  title: "十二地支配对",
  description: "掌握地支的配对关系",
  questions: [
    {
      id: 1,
      type: "single_choice",
      question: "子与哪个地支配对？",
      options: ["丑", "寅", "卯", "辰"],
      correctAnswer: "丑",
      explanation: "子与丑配对，属于六合之一。"
    },
    {
      id: 2,
      type: "fill_in_blank",
      question: "六合配对：子__、寅__、辰__。",
      correctAnswer: "丑,亥,酉",
      explanation: "六合：子丑、寅亥、辰酉、巳申、午未、卯戌"
    }
  ]
}
```

## ⚠️ 注意事项

1. **课程编号 (id)**：
   - 必须唯一
   - 按顺序递增（1, 2, 3, 4...）
   - 不能重复

2. **题目编号 (questions[].id)**：
   - 每个课程内的题目编号从1开始
   - 可以不是连续的（但建议连续）

3. **填空题答案**：
   - 多个答案用**英文逗号**分隔：`"答案1,答案2,答案3"`
   - 答案顺序可以任意（系统会自动匹配）
   - 标点符号会被忽略

4. **单选题选项**：
   - 建议4个选项
   - `correctAnswer` 必须是 `options` 中的一个

5. **保存后**：
   - 刷新页面即可看到新课程
   - 新课程会自动出现在课程列表中
   - 如果前面的课程未完成，新课程会显示为锁定状态

## 🔧 批量添加技巧

如果需要添加多个课程，可以：

1. **准备课程数据**（可以用 Excel 或文本编辑器）
2. **按格式整理**成 JSON 格式
3. **复制到 data.js** 中
4. **检查格式**是否正确（注意逗号、引号等）

## 📝 当前课程列表

查看 `src/data.js` 文件，可以看到所有现有课程。新课程应该添加在数组末尾。

## 💡 提示

- 建议每个课程至少3-5道题目
- 题目类型可以混合使用（单选+填空）
- 解释说明要清晰易懂
- 可以先添加少量题目测试，后续再补充
