# 应用分享指南

## 方法一：局域网分享（推荐，适合同一WiFi下的设备）

### 步骤：

1. **确保开发服务器正在运行**：
   ```bash
   cd wuxing-app
   npm run dev
   ```

2. **查看服务器地址**：
   启动后，终端会显示类似信息：
   ```
   ➜  Local:   http://localhost:5173/
   ➜  Network: http://192.168.x.x:5173/
   ```

3. **获取你的IP地址**（如果终端没显示）：
   ```bash
   # macOS/Linux
   ifconfig | grep "inet " | grep -v 127.0.0.1
   
   # Windows
   ipconfig
   ```

4. **分享链接**：
   - 将 `Network` 地址（如 `http://192.168.0.124:5173`）分享给同一WiFi下的其他人
   - 他们在手机或电脑浏览器中输入这个地址即可访问

### 注意事项：
- ✅ 手机和电脑必须在同一WiFi网络
- ✅ 开发服务器必须保持运行
- ✅ 防火墙可能阻止端口，需要允许5173端口

## 方法二：构建生产版本并部署

### 步骤：

1. **构建生产版本**：
   ```bash
   cd wuxing-app
   npm run build
   ```
   这会生成 `dist` 文件夹，包含所有静态文件。

2. **部署选项**：

   **选项A：使用 Vercel（免费，最简单）**
   - 访问 https://vercel.com
   - 注册/登录账号
   - 点击 "New Project"
   - 上传 `dist` 文件夹或连接GitHub仓库
   - 自动获得一个公共URL（如 `your-app.vercel.app`）

   **选项B：使用 Netlify（免费）**
   - 访问 https://netlify.com
   - 注册/登录账号
   - 拖拽 `dist` 文件夹到部署区域
   - 自动获得一个公共URL（如 `your-app.netlify.app`）

   **选项C：使用 GitHub Pages（免费）**
   - 将代码推送到GitHub仓库
   - 在仓库设置中启用GitHub Pages
   - 选择 `dist` 文件夹作为源
   - 获得URL（如 `username.github.io/repo-name`）

   **选项D：使用自己的服务器**
   - 将 `dist` 文件夹上传到服务器
   - 配置Web服务器（如Nginx、Apache）
   - 通过域名或IP访问

---

## 方法三：使用 ngrok（创建公共链接，无需部署）

### 步骤：

1. **安装 ngrok**：
   - 访问 https://ngrok.com
   - 注册免费账号并下载
   - 解压并添加到系统PATH

2. **启动开发服务器**：
   ```bash
   cd wuxing-app
   npm run dev
   ```

3. **在另一个终端运行 ngrok**：
   ```bash
   ngrok http 5173
   ```

4. **分享链接**：
   - ngrok会显示一个公共URL（如 `https://xxxx.ngrok.io`）
   - 这个链接可以从任何地方访问
   - 分享这个链接即可

### 注意事项：
- ⚠️ 免费版ngrok链接每次重启都会变化
- ⚠️ 开发服务器必须保持运行
- ✅ 适合临时分享和测试

---

## 方法四：打包为移动应用（PWA）

应用已经配置为PWA（Progressive Web App），可以：

1. **在手机上添加到主屏幕**：
   - 打开应用
   - iOS Safari：点击分享 → "添加到主屏幕"
   - Android Chrome：菜单 → "添加到主屏幕"

2. **分享PWA链接**：
   - 使用上述任一方法获得公共链接
   - 其他人访问后也可以添加到主屏幕
   - 就像原生应用一样使用

---

## 推荐方案

- **快速测试/演示**：使用方法一（局域网）或方法三（ngrok）
- **正式发布**：使用方法二（部署到Vercel/Netlify）
- **长期使用**：使用方法二（部署）+ 自定义域名

## 常见问题

**Q: 手机打不开链接？**
- 检查是否在同一WiFi
- 检查防火墙设置
- 尝试使用电脑IP地址而不是localhost

**Q: 如何让链接永久有效？**
- 使用方法二部署到Vercel/Netlify
- 免费版提供永久链接

**Q: 可以分享给不在同一WiFi的人吗？**
- 使用方法二（部署）或方法三（ngrok）
