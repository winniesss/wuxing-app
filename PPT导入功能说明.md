# PPT 导入功能说明

## 📋 当前状态

目前应用**不支持直接导入PPT文件**，但可以通过以下方式实现：

## 🎯 实现方案

### 方案一：手动转换（当前可用）

1. **从PPT提取内容**
   - 打开PPT文件
   - 复制题目和答案
   - 按照课程格式添加到 `src/data.js`

2. **使用模板快速添加**
   - 参考 `快速添加课程指南.md`
   - 复制模板，填入PPT中的内容

### 方案二：开发导入功能（需要开发）

可以开发以下功能：

#### 1. **PPT文件上传**
   - 在设置页面或管理页面添加上传功能
   - 支持 `.pptx` 文件格式
   - 解析PPT内容，提取题目和答案

#### 2. **Excel/CSV导入**
   - 支持Excel或CSV格式
   - 格式：题目 | 类型 | 选项 | 正确答案 | 解释
   - 一键导入生成课程

#### 3. **文本格式导入**
   - 支持特定格式的文本文件
   - 自动解析并转换为课程数据

## 🚀 快速实现方案（推荐）

### 创建一个简单的导入工具页面

可以在设置页面添加一个"导入课程"功能：

```javascript
// 功能包括：
1. 文件上传（支持 .pptx, .xlsx, .txt）
2. 内容解析
3. 预览和编辑
4. 确认导入
```

## 📝 PPT内容格式建议

如果要从PPT导入，建议PPT按以下格式组织：

### 格式示例：

**幻灯片1：课程信息**
- 标题：课程名称
- 描述：课程简介

**幻灯片2-N：题目**
- 题目类型（单选/填空）
- 问题内容
- 选项（如果是单选）
- 正确答案
- 解释说明

## 🔧 开发导入功能步骤

如果需要开发这个功能，需要：

1. **安装依赖**
   ```bash
   npm install mammoth  # 解析PPT/Word
   npm install xlsx     # 解析Excel
   ```

2. **创建导入组件**
   - `src/components/ImportCourse.jsx`
   - 文件上传
   - 内容解析
   - 数据预览

3. **添加到设置页面**
   - 在 `SettingsPage.jsx` 中添加导入入口

4. **数据验证和保存**
   - 验证格式
   - 保存到 `data.js` 或 localStorage

## 💡 临时解决方案

在导入功能开发完成前，可以：

1. **使用Excel整理**
   - 在Excel中整理题目
   - 按照固定格式填写
   - 手动复制到代码中

2. **使用文本模板**
   - 创建文本模板文件
   - 填写内容
   - 使用脚本转换

3. **批量添加工具**
   - 创建一个简单的表单页面
   - 输入题目信息
   - 自动生成代码格式
