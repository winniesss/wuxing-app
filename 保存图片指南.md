# 如何保存 Huskie 图片

## 保存 husky_happy.png

1. **找到图片文件位置**
   - 如果你有图片文件，请找到它的位置
   - 或者从设计工具/资源包中导出

2. **保存到正确目录**
   ```
   wuxing-app/public/assets/huskie-study/export/png/husky_happy.png
   ```

3. **验证文件**
   运行检查脚本：
   ```bash
   cd wuxing-app
   node check-huskie-assets.js
   ```

## 快速保存命令（如果你有图片文件）

```bash
# 如果图片在 Downloads 文件夹
cp ~/Downloads/husky_happy.png wuxing-app/public/assets/huskie-study/export/png/

# 如果图片在当前目录
cp husky_happy.png wuxing-app/public/assets/huskie-study/export/png/
```

## 需要的所有文件

请确保保存以下 6 个 PNG 文件：
- husky_happy.png ✅ (你提到的这个)
- husky_study.png
- husky_confused.png
- husky_sleep.png
- husky_dragon.png
- husky_crystal.png
