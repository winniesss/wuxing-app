# 五行天干地支背诵应用

一个帮助学习五行、天干、地支知识的交互式学习应用。

## 功能特点

- 📚 6个学习关卡，循序渐进
- 🎯 单选题和填空题两种题型
- 📊 学习进度跟踪
- 🗺️ 可视化学习地图
- 📱 响应式设计，支持手机和电脑
- 🎨 精美的UI设计

## 技术栈

- React 19
- Vite
- 纯CSS（无框架依赖）

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 部署

推荐使用 Vercel 一键部署：

1. 将代码推送到 GitHub
2. 在 Vercel 导入 GitHub 仓库
3. 自动部署完成
